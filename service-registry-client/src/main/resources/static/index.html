Welcome to Service Registry Debugger!

This service provides an easy way to lookup service information from Eureka.
You use it by the following URL: /lookup/serviceID where serviceID is the ID of the respective service as declared in Eureka.

For example, the following URLs show information about the respective registered services

  1. Catalog Maintenance Service
  2. User Service
  3. Checkout Service
  4. Web Shop Service
  5. Order Fulfillment Service
  6. The Service Registry Debugger itself
You can simply copy and paste these URLs into a hystrix dashboard.
